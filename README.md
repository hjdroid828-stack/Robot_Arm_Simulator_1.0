# 2-DOF Robot Arm Kinematics Simulator

## Overview
A Python-based 2-DOF Robot Arm Simulator with GUI, Forward/Inverse Kinematics, and Matplotlib visualization.

## Features
- Forward Kinematics
- Inverse Kinematics
- Robot Visualization
- Workspace Visualization
- CustomTkinter GUI
- Reset Function

## Technologies
- Python 3.12
- CustomTkinter
- NumPy
- Matplotlib

## Project Structure
- main.py
- gui.py
- kinematics.py
- visualization.py
